# 🌠 Meteor Storm - 超絶格好良い隕石落下

## 概要
壮観な隕石落下アニメーションシステムです。複数の隕石タイプ、カスタマイズ可能なパラメーター、そして没入感のある効果音を特徴とします。

## 機能

### ✨ 隕石タイプ
- 🔥 ファイアメテオ: 赤とオレンジの炎エフェクト
- ❄️ アイスメテオ: 青と水色の氷エフェクト
- 🌌 コズミックメテオ: 紫とピンクの宇宙エフェクト
- ⚡ プラズマメテオ: マゼンタと赤の電撃エフェクト
- 🌈 レインボーメテオ: 虹色の多彩なエフェクト

### 🎮 コントロール
- **隕石サイズ**: 1-5段階の大きさ調整
- **降下速度**: 1-10段階のスピード調整
- **隕石密度**: 1-10段階の密度調整
- **効果音**: ON/OFF切り替え
- **エピックモード**: 全パラメーター最大値での超迫力モード

### 🎯 統計表示
- 総隕石数: 降ってきた隕石の総数
- 破壊力: 隕石による累積ダメージ値

## 技術仕様

### アーキテクチャ
- **Canvas技術**: 3層Canvas構成（隕石、パーティクル、グロー）
- **オブジェクト指向設計**: MeteorStormクラスベース
- **リアルタイム物理演算**: 重力、軌跡、爆発エフェクト

### エフェクトシステム
- **軌跡レンダリング**: フェードアウト付きトレイル
- **パーティクル爆発**: 放射状パーティクル生成
- **グロー効果**: ブラー付きグロー層
- **グラデーション**: 放射状カラーグラデーション

### 音響システム
- **Web Audio API**: 手動音声生成
- **サウンドタイプ**: Whoosh、爆発音
- **動的音響**: 隕石サイズに応じた音量調整

## 使用方法

### 基本操作
1. 隕石タイプを選択
2. サイズ、速度、密度を調整
3. 「隕石嵐開始」ボタンをクリック
4. リアルタイムでアニメーションを楽しむ

### エピックモード
「エピックモード」ボタンで超迫力体験：
- 全パラメーター最大値
- ランダム隕石タイプ
- 画面シェイクエフェクト

## カスタマイズ

### CSS変数
```css
:root {
    --bg-dark: #0a0a0f;      /* 背景色 */
    --primary: #ff6b35;       /* プライマリ色 */
    --secondary: #f7931e;     /* セカンダリ色 */
    --accent: #ffd23f;        /* アクセント色 */
}
```

### 隕石タイプ追加
```javascript
// script.js内のmeteorTypesオブジェクトに追加
newType: {
    colors: ['#color1', '#color2', '#color3'],
    trailColor: 'rgba(r, g, b, 0.6)',
    glowColor: 'rgba(r, g, b, 0.3)',
    particleColors: ['#color1', '#color2']
}
```
